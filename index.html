花甲老头文章摘录

有一个螃蟹原理，大意是说：用敞口藤篮来装螃蟹，一只螃蟹很容易爬出来，多装几只后，就没有一只能爬出来了，不因为别的原因，就因为相互扯后腿的缘故。

在人性里，也有类似的现象，就是一个人绝大多数招数，都会用在周围人身上

兄弟几个人，如果都富裕，那么关系就很难闹掰，即使一个人有个更大的突破，周围人也不会过于嫉妒，因为自己也还不错。

还是兄弟几个人，如果吃饭都难，突然有个人富裕起来，那么其他人会想办法给他折腾穷。

仓廪食而知礼节，衣食足而知荣辱
可见，越往下沉沦，日子越艰难，因为人性如此，他们就像一双双巨大的手，给你牢牢抓住

而且往上去闯，人会有很大的优势，因为阎王好斗，小鬼难缠，因为你和阎王不是一个级别，人家不会为难你，这一点最关键。

大家记住，真正难为你的，就是你身边的差不多的人，还有比你高一点的人，其他人会给你的宽容大很多，而你的路也会因为遇到这些人而变宽

很多人都在追求财富，不过只是停留在财富的层级去追求财富，其实任何答案，都在高一个维度。

比如做生意求财，你卖的是产品么？你卖的是口碑,服务, 阶层差异以及生态。
做工程也好，弄项目也罢，只关注这件事情本身，也是不对的，应该是更高一层的比如人情世故等等.

财富是工具，不是结果，财富是实现价值的工具而已，而结果一定是自己追求的美好体验。

很多人本末倒置了，认为人的目标就是追逐财富，而这种满脑子是钱，反而不会有大钱降临，还耽误了美好生活。

所以，我一直不太建议年轻人攒钱，钱不是攒出来的，而是因为机遇流向你这里的

git+hexo windows环境搭建github.io博客

[toc]

windows环境搭建github.io博客

软件安装

安装nodejs(包含了npm工具)

Download and install Windows Installer (.msi) from

安装hexo

npm install hexo-cli -g

安装git

git download

配置git的ssh公钥
git config --global user.name zd  
git config --global user.email dkd1168@126.com  

ssh-keygen -t rsa -C dkd1168@126.com

将公钥id_rsa.pub的内容拷贝到github页面的sshkeys

安装hexo的git推送器

npm install hexo-git-backup --save

hexo 创建项目

hexo init zd_blog_test  
cd zd_blog_test & npm install

确认项目是否成功, 启动本地服务器, 访问http://localhost:4000 

hexo server

hexo部署到github.io

hexo有3个作用

  1. hexo new filename 生成md文件
  2. hexo g 将md文件转为html
  3. hexo d 将html文件推送到github.io
  4. 2,3可以合为一步hexo g –d

创建github.io账号

给仓库起个名称，注意！注意！注意！

仓库名称格式： 用户名.github.io 

https://github.com/hclt1168/hclt1168.github.io

配置hexo的_config.yml

# DeploymentV  
## Docs: https://hexo.io/docs/one-command-deployment  
deploy:  
  type: git  
  repo: git@github.com:hclt1168/hclt1168.github.io.git  
  branch: master  
  message: update

发布到github.io

  1. 新建博客
    hexo new zd-fisrt-blog

找到zd_blog_text/source/_posts/zd-fisrt-blog.md, 使用vscode打开并且编辑 


---  
title: zd-first-blog  
date: 2024-10-23 11:37:15  
tags: blog1  
---  
### 收入的实质  

收入和努力没关系，和你创造的社会价值有关系  
解决社会问题  
满足社会需求  
创造社会价值  
对人对企业都一样
  1. 生成html文件
    hexo g
  2. 上传到github
    hexo d
  3. 查看最终效果 zd-first-blog/
